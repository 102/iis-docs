**Applies to:** Windows Store apps only

## Syntax

``` csharp

[VersionAttribute()]
public sealed class HttpCookie : IHttpCookie
```

``` c++

[VersionAttribute()]
public ref class HttpCookie sealed : IHttpCookie
```
