# MoveNext Method

**Applies to:** Windows Store apps only

Move to the next chunk.

## Syntax

``` csharp
void MoveNext()
```

``` c++
void MoveNext()
```

``` jscript
function MoveNext()
```

## Return Value

If the method succeeds, it returns S\_OK. Otherwise, it returns an HRESULT error code.
