## Remarks

This class is abstract (MustInherit in Visual Basic) and must be inherited. This class and inheritors represent deployment HTTP requests. These requests are wrapped in the [DeploymentAgentHandler](deploymentagenthandler-class-microsoft-web-deployment.md) and passed to a [DeploymentAgent](deploymentagent-class-microsoft-web-deployment.md) to be processed.

## Thread Safety

Any public static (Shared in Visual Basic) members of this type are thread safe. Any instance members are not guaranteed to be thread safe.
