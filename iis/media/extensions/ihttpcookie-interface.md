# IHttpCookie Interface

**Applies to:** Windows Store apps only

Represents the HttpCookie interface.

## Syntax

``` csharp
[GuidAttribute(, , , , , , , , , , )]
[VersionAttribute()]
public interface IHttpCookie
```

``` c++
[GuidAttribute(, , , , , , , , , , )]
[VersionAttribute()]
public interface class IHttpCookie
```

``` jscript
public interface IHttpCookie
```
