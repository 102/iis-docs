# Manifest.IsLive Property

**Applies to:** Windows Store apps only

Gets the manifest 'isLive' field.

## Syntax

``` csharp
public bool IsLive { get; }
```

``` c++
public:
virtual property bool IsLive {
bool get () sealed;
}
```

``` jscript
final function get IsLive () : boolean
```

## Property Value

The manifest 'isLive' field.
