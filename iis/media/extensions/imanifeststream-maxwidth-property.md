# IManifestStream.MaxWidth Property

**Applies to:** Windows Store apps only

Gets the stream 'MaxWidth' field.

## Syntax

``` csharp
uint MaxWidth { get; }
```

``` c++
property unsigned int MaxWidth {
unsigned int get ();
}
```

``` jscript
function get MaxWidth () : uint
```

## Property Value

The stream 'MaxWidth' field.
