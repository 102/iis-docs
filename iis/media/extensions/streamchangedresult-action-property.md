# StreamChangedResult.Action Property

**Applies to:** Windows Store apps only

Gets the action of the stream.

## Syntax

``` csharp
public StreamChangedAction Action { get; }
```

``` c++
public:
virtual property StreamChangedAction Action {

}
```


## Property Value
