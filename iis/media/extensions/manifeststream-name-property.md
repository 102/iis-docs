# ManifestStream.Name Property

**Applies to:** Windows Store apps only

Gets the stream 'Name' field.

## Syntax

``` csharp
public string Name { get; }
```

``` c++
public:
virtual property String^ Name {
String^ get () sealed;
}
```

``` jscript
final function get Name () : String
```

## Property Value

The stream 'Name' field.
