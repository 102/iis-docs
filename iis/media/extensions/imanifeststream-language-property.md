# IManifestStream.Language Property

**Applies to:** Windows Store apps only

Gets the stream 'Language' field.

## Syntax

``` csharp
string Language { get; }
```

``` c++
property String^ Language {
String^ get ();
}
```

``` jscript
function get Language () : String
```

## Property Value

The language of stream.
