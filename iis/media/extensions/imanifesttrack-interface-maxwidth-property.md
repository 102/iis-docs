# IManifestTrack Interface.MaxWidth Property

**Applies to:** Windows Store apps only

Gets the track 'MaxWidth' field.

## Syntax

``` csharp
uint MaxWidth { get; }
```

``` c++
property unsigned int MaxWidth {
unsigned int get ();
}
```

``` jscript
function get MaxWidth () : uint
```

## Property Value

The track 'MaxWidth' field.
