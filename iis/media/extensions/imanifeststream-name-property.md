# IManifestStream.Name Property

**Applies to:** Windows Store apps only

Gets the stream 'Name' field.

## Syntax

``` csharp
string Name { get; }
```

``` c++
property String^ Name {
String^ get ();
}
```

``` jscript
function get Name () : String
```

## Property Value

The stream 'Name' field.
