# INamedContextContainer Interface
Provides an interface for binding and retrieving a named context.  
  
## Syntax  
  
```cpp  
class INamedContextContainer  
```  
  
## Methods  
 The following table lists the methods exposed by the `INamedContextContainer` interface.  
  
|Name|Description|  
|----------|-----------------|  
|[GetNamedContext](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/inamedcontextcontainer-getnamedcontext-method.md)|Retrieves a stored context given its name.|  
|[SetNamedContext](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/inamedcontextcontainer-setnamedcontext.md)|Binds stored context with a specified name.|  
  
## Derived Classes  
 This interface contains no derived classes.
