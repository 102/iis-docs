# IAppHostConfigFile::FilePath Property
Gets the file system path of the configuration file that this [IAppHostConfigFile](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/iapphostconfigfile-interface.md) interface represents.  
  
## Syntax  
  
```cpp  
HRESULT get_FilePath(  
   [out,  
   string,  
   retval] BSTR* pbstrFilePath  
);  
```  
  
#### Parameters  
 `pbstrFilePath`  
 A pointer to a `BSTR` that contains the file system path of the configuration file.  
  
## Return Value  
 An `HRESULT`. Possible values include, but are not limited to, those in the following table.  
  
|Value|Description|  
|-----------|-----------------|  
|S_OK|Indicates that the operation was successful.|
