# IHttpConnectionStoredContext Interface
Defines context information for connection-stored data.  
  
## Syntax  
  
```cpp  
IHttpConnectionStoredContext : public IHttpStoredContext  
```  
  
## Methods  
 The following table lists the methods exposed by the `IHttpConnectionStoredContext` interface.  
  
|Name|Description|  
|----------|-----------------|  
|[NotifyDisconnect](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/ihttpconnectionstoredcontext-notifydisconnect.md)|Notifies the caller that an established connection was disconnected.|  
  
## Derived Classes  
 This interface contains no derived classes.
