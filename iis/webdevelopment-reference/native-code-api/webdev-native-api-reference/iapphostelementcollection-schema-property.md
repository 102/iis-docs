# IAppHostElementCollection::Schema Property
Returns the schema assigned to the current collection.  
  
## Syntax  
  
```cpp  
HRESULT get_Schema(  
   [out,  
   retval] IAppHostCollectionSchema** ppSchema  
);  
```  
  
#### Parameters  
 `ppSchema`  
 A pointer to a pointer for an [IAppHostCollectionSchema](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/iapphostcollectionschema-interface.md) interface.  
  
## Return Value  
 An `HRESULT`. Possible values include, but are not limited to, those in the following table.  
  
|Value|Description|  
|-----------|-----------------|  
|S_OK|Indicates that the operation was successful.|  
|E_FAIL|Indicates that the operation was not successful.|
