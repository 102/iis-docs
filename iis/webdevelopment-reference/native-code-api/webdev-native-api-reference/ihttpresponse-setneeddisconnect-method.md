# IHttpResponse::SetNeedDisconnect Method
Resets the socket after the response is complete.  
  
## Syntax  
  
```cpp  
virtual VOID SetNeedDisconnect(  
   VOID  
) = 0;  
```  
  
#### Parameters  
 This method takes no parameters.  
  
## Return Value  
 `VOID`.  
  
## Remarks  
 The `SetNeedDisconnect` method requests that the socket be disconnected after the server finishes processing the current request. This method differs from the [IHttpResponse::ResetConnection](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/ihttpresponse-resetconnection-method.md) method, which terminates the socket connection immediately.
