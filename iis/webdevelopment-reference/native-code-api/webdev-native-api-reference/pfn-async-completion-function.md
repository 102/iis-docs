# PFN_ASYNC_COMPLETION function
Defines the asynchronous completion function prototype.  
  
## Syntax  
  
```cpp  
typedef REQUEST_NOTIFICATION_STATUS(WINAPI* PFN_ASYNC_COMPLETION)(  
   IHttpContext3* pHttpContext,  
   IHttpCompletionInfo2* pCompletionInfo,  
   VOID* pvCompletionContext  
);  
```  
  
#### Parameters  
 `pHttpContext`  
 A pointer to an [IHttpContext3](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/ihttpcontext3-interface.md) interface.  
  
 `pCompletionInfo`  
 A pointer to an [IHttpCompletionInfo2](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/ihttpcompletioninfo2-interface.md) interface.  
  
 `pvCompletionCointext`  
 Points to the completion context information for the operation.
