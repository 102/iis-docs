# IMapHandlerProvider::SetScriptName Method
This method is not supported.  
  
## Syntax  
  
```cpp  
virtual HRESULT SetScriptName(  
   PCWSTR pszScriptName,  
   DWORD cchScriptName  
) = 0;  
```  
  
#### Parameters  
 `pszScriptName`  
 Not supported.

 Not supported.  
  
## Return Value  
 Not supported.  
  
## Remarks  
 This method is not supported.
