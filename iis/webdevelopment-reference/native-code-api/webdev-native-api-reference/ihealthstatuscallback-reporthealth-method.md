# IHealthStatusCallback::ReportHealth Method
Allows a protocol manager to report its current health status in response to a query from the [IPmHealthAndIdleMonitor::CheckHealth](../../../webdevelopment-reference\native-code-api\webdev-native-api-reference/ipmhealthandidlemonitor-checkhealth-method.md) method.  
  
## Syntax  
  
```cpp  
virtual HRESULT ReportHealth(  
   BOOL fHealthy  
) = 0;  
```  
  
#### Parameters  
 `fHealthy`  
 `true` to indicate that the protocol manager is healthy; otherwise, `false`.  
  
## Return Value  
 An `HRESULT`. Possible values include, but are not limited to, those in the following table.  
  
|Value|Description|  
|-----------|-----------------|  
|S_OK|Indicates that the operation was successful.|  
  
## Remarks  
 Protocol managers are responsible for monitoring and reporting their own health status.
